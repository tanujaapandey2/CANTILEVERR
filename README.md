# CANTILEVERR
# Text Detection and Extraction using OCR (Flask App)

This project is a simple web application that allows users to upload an image and extract text from it using Tesseract OCR.

## Features
- Upload image
- Extract text using OCR
- Display extracted text
- Simple Flask UI

## Tech Stack
- Python
- Flask
- Tesseract OCR
- HTML/CSS

## Setup Instructions

1. Install Tesseract OCR:
   https://github.com/tesseract-ocr/tesseract

2. Clone repository:
   git clone https://github.com/your-username/text-detection-ocr-flask.git

3. Install dependencies:
   pip install -r requirements.txt

4. Run app:
   python app.py

5. Open browser:
   http://127.0.0.1:5000
